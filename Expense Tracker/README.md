# 💰 Expense Tracker

A responsive Expense Tracker built using HTML, CSS, and JavaScript.

## Features

- Add Income
- Add Expense
- Delete Transactions
- Automatic Balance Calculation
- Local Storage Support
- Responsive UI

## Technologies

- HTML5
- CSS3
- JavaScript
- Local Storage

## Folder Structure

```
06-Expense-Tracker/
│── index.html
│── style.css
│── script.js
│── README.md
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
