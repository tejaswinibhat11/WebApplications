# 🧮 Simple Calculator

## 📌 Description
A responsive calculator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Percentage
- Decimal Support
- AC (Clear)
- DEL (Delete Last Character)
- Responsive Design

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Folder Structure
```
01-Simple-Calculator/
│── index.html
│── style.css
│── script.js
│── README.md
```

## ▶️ How to Run
1. Download or clone the repository.
2. Open `index.html` in your browser.
