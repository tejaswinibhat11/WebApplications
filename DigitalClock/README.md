# ⏰ Digital Clock

## Description
A responsive Digital Clock built using HTML, CSS, and JavaScript.

## Features
- Live Clock
- AM / PM
- Current Date
- Current Day
- Responsive UI
- Glassmorphism Design

## Technologies
- HTML5
- CSS3
- JavaScript

## Run
Open `index.html` in your browser.
